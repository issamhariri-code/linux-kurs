Övningsuppgift 6.16
exit status = avslutningskod från senaste kommando (0=lyckades)
du kan se exit status för senaste kommando genom: echo $?
en exit status kan ha 0 om det gick bra eller 1-255 vid misslyckande
för att se att det gick bra kan du köra >
grep skol fil.txt && "Sko fanns!" > Sko fanns! printas ut då gick det bra (0)
