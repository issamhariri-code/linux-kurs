Övningsuppgift 2.2
För att installera ett paket med apt använder man kommandot:
sudo apt install <paketnamn>
Svar: Ja det kräver root privilegier eftersom man gör ändringar i systemet när sudo
används. Vid kommandot krävs oftast lösenordsverifiering.

Övningsuppgift 2.12
sudo apt update - uppdaterar själva katalogen för programvaran och hämtar all ny info,
görs dock inga uppdateringar utan bara själva katalogen.
sudo apt upgrade - installerar själva uppdateringen av programvaran(försiktigare vers)
sudo apt full-upgrade är för fullständig uppgradering där systemet får ta bort paket
om det krävs för att lösa beroenden.
Övningsuppgift 2.13
/dev innehåller specialfiler, enhetsfiler som står för hårdvara och virtuella enheter 
i systemet. Där måste det finnas filer som /dev/null /dev/zero /dev/tty.
/dev/null = slänger bort data, /dev/zero = ger nollor, /dev/tty = terminalen.
